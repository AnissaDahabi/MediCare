## À propos

MedicalCare est conçu pour simplifier la gestion médicale via une interface simple et sécurisée. L’application suit le modèle MVC (Modèle-Vue-Contrôleur) et permet un contrôle des accès selon le rôle de l’utilisateur.

---

## Rôles et permissions

| Rôle  | Permissions principales |
|-------|------------------------|
| **Admin** | Ajouter, modifier et supprimer des patients et médecins. Gérer consultations et prescriptions. Modifier toutes les informations du système. |
| **User**  | Consulter les informations disponibles : patients, médecins, consultations, prescriptions. Pas de droits d’ajout, modification ou suppression. |

---

## Fonctionnalités

- Gestion des patients et médecins (CRUD pour les Admins).  
- Gestion des consultations et prescriptions.  
- Consultation des informations pour les Users.  
- Interface web intuitive.  
- Contrôle d’accès par rôle.

---

## Technologies utilisées

- **Langage** : PHP  
- **Base de données** : MySQL  
- **Frontend** : HTML, CSS  
- **Architecture** : MVC (Modèle-Vue-Contrôleur)  
- **Hébergement** : AlwaysData
